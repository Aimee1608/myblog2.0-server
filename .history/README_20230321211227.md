### 接入微博授权登录

1. 先在微博开放平台完善个人信息，并做个人身份认证
2. 创建我的应用，完善应用信息，提交审核
3. 填写 oauth2 的授权地址信息
4. 审核通过前，填写测试账号调试

- 文档地址：https://open.weibo.com/wiki/%E6%8E%88%E6%9D%83%E6%9C%BA%E5%88%B6#.E6.8E.88.E6.9D.83.E6.9C.89.E6.95.88.E6.9C.9F
- 我的应用中心：https://open.weibo.com/apps
